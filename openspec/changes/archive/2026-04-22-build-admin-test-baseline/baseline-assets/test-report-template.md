# 测试报告模板

## 1. 使用边界

- 首版全量测试基线构建：本模板只用于定义报告字段与结果归档方式，不生成真实业务测试结论
- 后续迭代差异分析：本模板必须按 `DIFF / RULE / TC / AUTO` 输出结果、未覆盖项和残余风险
