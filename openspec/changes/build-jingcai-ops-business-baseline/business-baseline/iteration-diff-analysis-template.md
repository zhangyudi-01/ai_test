# 迭代差异分析模板

用于在 `BASE-JCOPS-2026-04-22-V1` 基线之上分析后续需求变化。

## 一、基本信息

| 字段 | 填写要求 |
| --- | --- |
| 差异分析版本 | `DIFF-YYYYMMDD-XX` |
| 引用基线版本 | 必填，引用固定 `BASE-*` 版本 |
| 变更主题 | 必填，写业务变更名称 |
| 变更来源 | 必填，PRD 修订、原型更新、接口补充、运营反馈等 |
| 来源回指 | 必填，章节/标题/链接/附件 |
| 分析人 | 必填 |
| 分析日期 | 必填 |
| 变更状态 | `分析中 / 待确认 / 已冻结 / 已回写基线` |

## 二、差异项清单

| 差异ID | 变更类型 | 受影响资产ID | 原行为 | 新行为 | 影响模块/页面/API | 来源类型 | 来源回指 | 是否回写基线 | 备注 |
| --- | --- | --- | --- | --- | --- | --- | --- | --- | --- |
| `DIFF-001` | 新增 / 修改 / 删除 / 澄清 | `RULE-*` / `PAGE-*` / `OBJ-*` / `ACT-*` | 必填，引用基线中的旧行为 | 必填，写新行为 | 必填 | `PRD章节 / 原型链接 / 外部附件 / 接口表` | 必填 | `是 / 否 / 待定` | 可写 `依赖外部原型补充` |

## 三、影响分析

| 影响维度 | 影响说明 | 受影响资产ID |
| --- | --- | --- |
| 模块边界 | 是否改变模块职责或边界 | `MOD-*` |
| 页面/交互 | 是否新增、删除或修改页面入口、字段、动作 | `PAGE-*` |
| 对象/字段 | 是否新增、删除或修改对象字段、唯一性、枚举、状态 | `OBJ-*` / `FIELD-*` |
| 业务规则 | 是否新增、删除或修改规则 | `RULE-*` |
| 状态/流程 | 是否改变状态流转或关键流程顺序 | `STATE-*` / `FLOW-*` |
| 外部依赖/API | 是否新增外部依赖、接口或文件流转 | `EXT-*` / `ACT-*` / `FILE-*` |
| 权限 | 是否改变角色和权限包授权范围 | `ROLE-*` / `AUTH-*` |

## 四、基线更新决策

| 决策项 | 填写要求 |
| --- | --- |
| 是否立即回写基线 | `是 / 否` |
| 若否，原因 | 如：方案未冻结、外部原型未补齐、接口未定 |
| 若是，回写目标 | 列出需要更新的基线文件 |
| 回写完成标记 | `未开始 / 已完成` |

## 五、未决问题

| 问题ID | 问题描述 | 是否阻塞 | 计划补充来源 |
| --- | --- | --- | --- |
| `Q-NEW-001` | 写本次变更新增的未决问题 | `是 / 否` | 原型 / PRD / 接口 / 运营确认 |
