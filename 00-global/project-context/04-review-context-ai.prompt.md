### 1. 粒度控制检查

### 2. 需求拆分检查
**问题**：是否提前做了原子需求拆解？

**检查要点**：
- [ ] 是否逐条拆解需求

### 3. 测试设计检查
**问题**：是否提前生成了测试点、测试方案、测试用例？

**检查要点**：
- [ ] 是否生成测试点清单

### 4. 资料权威性检查
**问题**：是否把历史资料当成当前需求依据？

**检查要点**：
- [ ] 是否用历史需求推测当前需求

### 5. 权威依据检查
**问题**：是否明确当前需求是最高依据？

**检查要点**：
- [ ] 是否明确标注当前需求为最高权威

### 7. 不确定性标记检查
**问题**：是否明确不确定内容要进入待确认？

**检查要点**：
- [ ] 不确定内容是否使用【待确认】【待补充】【存疑】标记
